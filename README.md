# RV4028: IO module firmware

This is the firmware for the [RV4028](https://github.com/MichaelBell/RV4028) IO Module.

It uses the RP2350 in RISC-V mode, and by default expects the RP2350 RISC-V toolchain to be in `$PICO_SDK_PATH/../riscv-toolchain`.  Get the toolchain from [pico-sdk-tools](https://github.com/raspberrypi/pico-sdk-tools/releases).
